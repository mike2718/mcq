# 随机选择题选项生成器

使用`/dev/urandom`提供的熵打乱ABCD四个字符，是[《A类业余无线电台操作技术能力考试攻略本》](https://github.com/mike2718/ham)写作过程中的副产物。

## 执行例子

```
mike@hp2540p mcq $ ./mcq.yash                                          git@main
打乱前  A B C D
打乱后  A D B C
正确选项        B

```

## 依赖

- [yash](https://yash.osdn.jp/index.html)
